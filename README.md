## 💻 Local Development

1. Clone the repository
```bash
git clone https://github.com/pragya238/DVA-portfolio.git
cd DVA-portfolio
```

2. Install dependencies
```bash
npm install
```

3. Start the development server
```bash
npm run dev
```
